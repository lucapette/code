# Web

Monorepo containing my websites built with Hugo and Go templates.

## Sites

- [main](/main) - [lucapette.me](https://lucapette.me)
- [notes](/notes) - [notes.lucapette.me](https://notes.lucapette.me)

## Tech Stack

- Static sites generated with Hugo (Go templates)
- CSS processed with PostCSS
- Hosted on Netlify/Vercel

## Structure

- `/main` - Main website content and layouts
- `/notes` - Notes and blog content
- `/assets` - SCSS/CSS and other assets

## Development

1. Install Hugo extended version
2. Clone repository
3. Run `hugo server` for local development
