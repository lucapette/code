# Kafka Streams vs TypeStream

Companion repository comparing Apache Kafka Streams and TypeStream for stream processing applications.

## Overview

Sample applications demonstrating the differences between Kafka Streams and TypeStream frameworks. Created for the comparative analysis article exploring both technologies.

## Features

- Equivalent stream processing logic implemented in both frameworks
- Side-by-side comparison of API differences
- Performance and usability considerations
- Real-world use case implementations

## Tech Stack

- Kotlin
- Apache Kafka & Kafka Streams
- TypeStream
- Gradle build system

## Structure

- `kafka-streams/` - Implementation using Apache Kafka Streams
- `typestream/` - Implementation using TypeStream
- Shared test data and configurations
